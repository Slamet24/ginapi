Struktur folder dan folder yang boleh diakses :
- controllers -> Folder untuk controller
- model -> Folder untuk model (koneksi database,query,dll)
- public -> Folder berisi index.php (untuk membuat sebuah route get/post)

Sebelum memakainya mohon sudah menginstall composer
Lalu, buka terminal di folder yang sudah kamu clone
Dan jalankan perintah composer update

Jika masih ada masalah, mohon chat digrup.
